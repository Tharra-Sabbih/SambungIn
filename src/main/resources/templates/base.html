Frequently Asked Question!



Token
Bagaimana caranya mendapatkan Secret Code?

Karena Secret Code merupakan Token dan bersifat personal, maka penulis tidak bisa memberikannya secara publik. Silahkan PC Tharra Sabbih (kontak ada di grup regional) untuk meminta tokenmu agar dapat mengakses fitur berinteraksi dengan KsaTiara!

Token
Apakah Token ini bersifat permanent atau temporary?

Token bersifat permanent. Setiap anggota keluarga KsaTiara memiliki satu token yang tetap dan tidak berubah-ubah.

Token
Apakah Tokenku bersifat rahasia?

Ya. Ibarat akun sebagai identitas, Token di SambungIn juga digunakan untuk memastikan bahwa kamu adalah kamu XD !



Pesan
Aku bisa ngirim pesan ke siapa aja?

Semua keluarga KsaTiara! Termasuk Ka Mute dan Bang Kabul, kita bisa kirim pesan dengan nama maupun anonym!

Pesan
Bagaimana cara mengirim pesan secara anonym?

Masukkan Secret Code (Token) yang tersedia untuk publik bernama ANONYMUS dengan token: 1593205784, lalu tinggal kirimkan pesan kepada siapapun yang kamu mau!

Pesan
Siapa saja yang bisa mengakses pesan yang ditujukan untuk seseorang?

Tenangg, pesanmu hanya dapat dibaca oleh seorang yang tertuju! Jadi jangan ragu untuk menyampaikan isi hatii xixi
